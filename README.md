# 🎯 重要用語クイズマスター 利用マニュアル

このツールは、IT専門用語を効率的に学習・暗記するためのクイズアプリケーションです。

## 🚀 基本的な使い方

### 1. 学習を始める
* **章の選択**: 画面上部のプルダウンから学習したい章を選択します。
* **出題方向**: 「用語 → 意味」または「意味 → 用語」を切り替えられます。
* **モード選択**:
    * **全問**: 選択した章のすべてのカードを出題します。
    * **NGのみ（苦手集中）**: 学習履歴（`v_stats`）を参照し、「正解2回未満」または「不正解歴あり」の問題だけを抽出して出題します。※章全体が出題されるわけではなく、苦手な問題だけに絞り込まれます。

### 2. 回答操作
* **マウス操作**: 4つの選択肢から正解と思うものをクリックします。
* **キーボード操作（推奨）**:
    * **数字キー [1]〜[4]**: 選択肢の選択。
    * **[Space] または [Enter]**: 回答後の「次へ進む」操作。
    * **[←] / [→]**: 前後の問題への移動（回答後）。

## 🔍 解説機能の仕組み (`parseExtra` の仕様)

解説文は、1つの文字列を特定のキーワードで自動分割してタブ表示しています。

### データの記述ルール
`extraExplain` フィールドに以下のキーワードを**行頭**に記述することで、正しく分割されます。

| 項目名       | キーワード  | 表示される内容                           |
| :----------- | :---------- | :--------------------------------------- |
| **ポイント** | `ポイント:` | 用語の核心、試験に出る重要部分。         |
| **具体例**   | `例:`       | 理解を助ける具体的な計算式やイメージ。   |
| **暗記法**   | `暗記:`     | 短いフレーズや覚え方のコツ。             |
| **注意点**   | `ひっかけ:` | 間違いやすいパターンや類似用語との違い。 |

> **ロジックのポイント**: 文章の途中にある「例：」には反応せず、行の先頭にあるキーワードのみを区切りとして認識します。これにより「桁落ち」のような記号を含む解説も欠損なく表示されます。

## 📊 学習管理と分析

### 進捗グラフ
「📊 グラフ」ボタンで、章ごとの習得状況を色分け表示します。
* **青（マスター）**: 2回以上正解し、苦手意識のない用語。
* **黄（学習中）**: 正解1回の用語。
* **赤（苦手）**: 不正解がある用語。
* **灰色（未着手）**: 未解答。

---

## 💡 学習効率を上げるコツ
1. **ピンポイント復習**: グラフで「赤」が多い章で「NGのみ」モードを使い、効率よく苦手を潰しましょう。
2. **高速回答**: 左手[1-4]、右手[Space]のホームポジションを覚えると、学習スピードが劇的に上がります。
